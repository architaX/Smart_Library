# Smart Library Analytics & Real-Time Notification System

A role-based Flask library management system with:
- Authentication
- Inventory management
- Automatic fine calculation
- Real-time notifications
- Analytics dashboard
- Personalized user notifications
- Python threading

## Tech Stack
- Python Flask
- SQLite
- HTML/CSS/JavaScript
- Chart.js

## Features
- Librarian dashboard
- User dashboard
- Inventory tracking
- Book issue-return system
- Fine automation
- Real-time alerts
- Analytics graphs

## Run Project

```bash
pip install -r requirements.txt
python app.py
